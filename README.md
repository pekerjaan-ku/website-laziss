# Lembaga Amil Zakat, Wakaf & Shodaqoh Khoiro Ummah

Website resmi lembaga amil zakat, wakaf, dan shodaqoh yang modern, elegan, dan penuh animasi.

## Fitur

- ✅ Animasi loading screen estetik
- ✅ Menggunakan AOS, Framer Motion, dan tsParticles
- ✅ Navbar responsif dengan menu: Home, Berita, Galery, Kontak
- ✅ Halaman Home dengan banyak section informatif
- ✅ Halaman Berita, Galery, dan Kontak lengkap
- ✅ Desain modern, responsive, dan mobile-friendly

## Cara Menggunakan

1. Unduh semua file ke folder lokal
2. Buka `index.html` di browser
3. Untuk pengembangan lebih lanjut, gunakan server lokal seperti Live Server (VS Code)

## Teknologi

- HTML5, CSS3, JavaScript
- Bootstrap 5
- AOS Animation
- Framer Motion
- tsParticles
- Font Awesome
